# WINMTR显IP信息版

## 特别说明：
* 源码来自：https://github.com/oott123/WinMTR

## 版本改进：
* 修复了没有纯真IP库qqwry.dat时软件奔溃的问题
* 32位版本兼容XP系统
* 界面优化
* 将IP和IP Information分开
* 加入Ping次数,默认为10000次，可自行修改
* 加入右键菜单（可复制IP，查看IP的Whois）
* 导出TXT和HTML时加入导出时间信息（当地时间）
